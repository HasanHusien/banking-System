A real-world banking system simulation built with React and Redux to demonstrate modern state management techniques.

This project compares two approaches for managing global state in React applications:

Redux (Classic Implementation)

Redux Toolkit (Modern Recommended Approach)

The goal of this project is to understand how Redux works internally and how Redux Toolkit simplifies Redux development.

 Features

Deposit money

Withdraw money

Request a loan

Pay loan

Create customer account

Global state management using Redux

Async operations (currency conversion API)

Comparison between Classic Redux and Redux Toolkit
